# AGENTS.md

## Project
- Name: `translatorwebsite`
- Stack: Node.js + Express + EJS
- Entry point: `server.js`

## Init
1. Install dependencies:
   `npm install`
2. Start the app:
   `npm start`
3. Open:
   `http://localhost:3000`

## Dev Notes
- Main server file: `server.js`
- Views: `views/`
- Static CSS: `CSS/`
- Upload directory used at runtime: `uploads/`

## Guardrails
- Do not commit secrets or API keys.
- Prefer environment variables for service credentials.
- Keep edits minimal and scoped to requested work.
